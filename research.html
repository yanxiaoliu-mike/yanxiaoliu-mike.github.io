Publication

Rate Region of Scheduling a Wireless Network with Discrete Propagation Delays
with Professor Shenghao Yang and Jun Ma
Accepted and to be published by INFOCOM 2021.

We study the link scheduling problem of wireless networks where signal propagation delays are multiples of certain time interval. The problem can be modeled as a character of the independent sets of periodic graphs, which have infinitely many vertices. We show that the rate region of scheduling a network can be achieved using collision-free, periodic schedules, and derive a graphical approach to explicitly characterize the rate region. In particular, a collision-free schedule can be equivalent to a path in a graph called the scheduling graph induced by the network collision profile and the propagation delays, and hence the rate region is equal to the convex hull of the rate vectors associated with the cycles of the scheduling graph, which have bounded length. With the maximal independent set problem as a special case, calculating the whole rate region is NP hard and also hard to approximate. By exploring a partial order on the paths, we derive an algorithm to calculate a subset of the rate region more efficiently.Our results are also of independent interest for periodic graphs. 

Forthcoming Papers

Sketch-based Per-flow Correlation Measurement Framework
Will submit to a top conference soon(IFIP 2021).

It is important to be able to answer queries in real-time and infer interesting patterns on-time. Hence we need to implement measurement tools on SRAM, which has limited available memory.  We design a sketch-based framework to passively store information of packets passed through and have special per-flow measurement architecture which has optimal memory use. We use mathematical schemes to reduce the noises thus resulting in a good enough estimation of correlation between any two data flows. It is expected to be submitted to IFIP 2021 soon. 

Regenerating Code
with Postdoctoral Fellow Ximing Fu
will submit to a top conference this term.

In a distributed storage system, a data file is distributed to storage nodes and can be decoded from any subset of the storage nodes of size achieving a certain threshold. Most existing constructions of regenerating codes are based on algebraic deign with finite field large enough, which requires expensive computational operations. We propose an efficient functional regenerating code only uses binary addition and shift operations in the coding, repair and decoding processes with low computational complexity.

Locally Repairable Codes
with Postdoctoral Fellow Ximing Fu
will submit to a top conference this term.

Distributed storage systems for large-scale applications typically use replication for reliability. One solution for reducing storage and bandwidth overhead is to use regenerating codes, or say, erasure codes. But there is one limitation: it needs high-repair cost during single node failure. We study the repair metric of locality and characterize an information theoretic tradeoff that binds together the locality, code distance, and storage capacity of each node. The family of these optimal locally repairable codes (LRCs) can still be improved efficiency if we use certain design of generator matrix and use binary operations instead of computation on large finite field.

Projects

BATched Sparse code(BATS) Protocol
with Professor Shenghao Yang
Hong Kong government's "Intelligent Lamp" project.

BATched Sparse code(BATS) is proposed for communication through multi-hop networks with packet loss. Based on advanced network coding techniques, BATS Protocol is proposed to meet the on-going and up-coming data communication demands of various network applications, including IoT, underwater networks, sensor networks and vehicular networks. Such network communications are characterized by large-scale, multi-hop wireless networks, where the existing network communication protocols, like TCP/IP, cannot fulfill.
